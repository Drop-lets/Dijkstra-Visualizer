# Dijkstra最短路径算法演示

数据结构与算法B课程大作业。基于 **Dijkstra 算法** 的最短路径可视化工具，使用 Python + Pygame 构建。支持有向图的交互式编辑和算法执行过程的逐步可视化演示。

---

## ✨ 功能特性

- **图编辑**：交互式创建/删除顶点和有向边，拖拽移动顶点，右键编辑边权重
- **随机图生成**：一键生成随机有向图，顶点均匀分散，边混合近邻与远端
- **Dijkstra 算法可视化**：
  - 自动播放（800ms/步）和手动步进
  - 顶点状态标记（未访问 / 当前 / 已访问 / 最短路径），光环高亮
  - 边松弛过程动画（橙色高亮正在松弛的边）
  - 最短路径高亮（绿色）
- **实时数据面板**：
  - 📊 距离表（带滚动条）
  - 📋 优先队列状态（带滚动条）
  - 📝 步骤详情（累积历史，最新步骤置顶，自动换行，带滚动条）
- **双向边权重可视化**：双向边的权重标签各自靠近源顶点，方向一目了然

## 🧠 核心算法

### Dijkstra 最短路径算法

本项目实现了经典的 **Dijkstra 算法**，使用**最小堆（heapq）** 作为优先队列，时间复杂度 O((V+E) log V)。

算法通过 Python Generator 逐步 `yield` 快照，与 Pygame 帧率解耦，支持正向步进、自动播放和步退查看。

### 课程知识点应用

| 知识点 | 应用位置 |
|--------|---------|
| **图 — 邻接表** | `models/graph.py` — `Graph` 类使用 `dict[str, list[Edge]]` 存储有向图 |
| **优先队列 / 最小堆** | `algorithms/dijkstra.py` — 使用 `heapq` 实现 O(log V) 的取最小距离操作 |
| **贪心算法** | Dijkstra 的核心思想：每次选择当前距离最小的未访问顶点 |
| **松弛操作** | 对每条出边检查 `dist[u] + weight < dist[v]`，若成立则更新 |
| **生成器模式** | 使用 `yield` 逐步输出 `DijkstraStep` 不可变快照，实现动画控制 |

### 算法步骤类型

| 步骤类型 | 说明 |
|---------|------|
| `init` | 初始化：起点距离=0，其余=∞ |
| `pop` | 从优先队列弹出当前距离最小的顶点，标记为已访问 |
| `relax_start` | 开始检查一条出边的松弛条件 |
| `relax_update` | 松弛成功：更新目标顶点距离和前驱 |
| `relax_check` | 松弛未成功：距离未改善 |
| `complete` | 算法完成：展示最短路径或报告不可达 |

## 🎨 随机图生成算法

`presets/sample_graph.py` 中的随机图生成：

1. **顶点布局**：拒绝采样逐个放置，保证任意两点间距 ≥ 动态最小距离 `max(90, √(面积 / (n × 1.8)))`。单个顶点最多尝试 400 次，失败取最佳位置。
2. **边生成**：每个顶点出边 2~4 条。候选目标按距离排序后分成近邻池和远端池各半，混合选取以避免边全部挤在一起。
3. **连通性保证**：BFS 检查从首个顶点出发的可达性，对不可达顶点随机添加补边。
